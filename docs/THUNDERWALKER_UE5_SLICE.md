# Thunderwalker UE5 First Slice

## Goal

Get James into engine quickly with the right combat target.

This slice is about proving:
- Call and Response combat
- Presence as the main encounter resource
- Medicine / Trick / Trial response families
- N64-ish readability and arena clarity

## Build first

- One Northern Forest arena
- One campfire staging area
- One Wendigo encounter
- One weapon, Spirit Lance recommended
- Player verbs:
  - light attack
  - heavy attack
  - dodge step
  - sidestep
  - brace
  - one Medicine action
  - one Trick action
  - one Trial action

## Do not overbuild yet

Skip for the first pass:
- deep inventory
- multiple bosses
- large procedural structure
- complex crafting trees
- huge animation sets
- card mechanics
- soulslike stamina obsession

## First encounter targets

### Wendigo calls
- slow convergence pressure
- fear aura / hunger pressure
- freeze breath spacing check
- heavy grab commitment

### Seeker responses
- Medicine to ground and stabilize Presence
- Trick to bait overcommitment
- Trial to brace and punish

## Arena notes

The arena should be readable in one screenshot:
- campfire edge
- frozen creek
- treeline
- one grounding landmark
- one phase-2 visibility complication

## Success test

If the prototype works, James should feel:
- "I read that attack correctly"
- "I answered it with the right kind of move"
- "This feels like a mythic duel"

Not:
- "I mashed harder"
- "I iframe-rolled everything"
- "I played cards at a boss"
